- Falls Fehlermeldungen der Art "Undefined control sequence \ifluatex when using document class..." auftauchen, im Internet recherchieren.
Es handelt sich vermutlich um ein zu installierendes Paket in der MikTex-Konsole

- TexStudio im Administrator-Modus starten --> F5 --> Pakete installieren lassen

- MikTex-Konsole aktuell halten
